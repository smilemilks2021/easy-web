Features

5 Auth Modes

auto, chromedp, browser, chrome, remote — choose the right mode for each site.

API Capture

Record all API requests from any page with URL pattern filtering and auto-save.

Authenticated Requests

Replay cookies for GET/POST/PUT/DELETE — like curl but with full browser auth.

Token Extraction

Pull tokens from headers, localStorage, or sessionStorage automatically.

Multi-Step Auth

Chain SSO flows with YAML config: browser login → token exchange → final auth.

Self-Update

easy-web selfupdate — always stay current, even on Windows.
